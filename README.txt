Description: This is a program that displays information about a room over time. It displays the temperature, humidity, light, co2, humidity ratio and occupancy status. 

Keys:
] - move right one tab (wraps around at end)
[ - move left one tab (wraps around at end)
g - toggle grid lines on and off.
